## Development

Install dependencies with:

```bash
mix setup
```

Preseed the database with mock data:

```bash
mix ecto.seed
```

Start Phoenix development server:

```bash
mix phx.server
```

Now you can visit [localhost:4000](http://localhost:4000) from your browser.
